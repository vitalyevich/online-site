Формы оплаты

Оплата наличными

Если отключите, пользователь не сможет
выбрать форму оплаты «наличными».

Оплата картой при получении

Для подключения других форм оплат (эквайринг, ерип)
обратитесь к вашему менеджеру.

Оплата онлайн

Сначала необходимо заключить договор с одной из платёжных систем (ArtPay, bePaid, BelAssist и др.), которая будет обрабатывать платежи. За подробностями обратитесь к вашему менеджеру.

Оплата переводом на карту

Курьер при доставке заказа сообщает клиенту номер карты. Далее клиент переводит на эту карту сумму заказа, удобным для него способом

Заказы на доставку курьерами

Снимите галку, если хотите принимать
заказы только на самовывоз.

Заказы на самовывоз

При самовывозе не добавляется стоимость доставки
и нет ограничений на минимальный заказ.

Drive-самовывоз

Самовывоз из специальных пунктов,
не выходя из машины.

Заказы в заведении

Посетитель заведения может сформировать заказ
и указать столик, за которым он сидит.

Уточнять сдачу

Поставьте галку, если при оплате наличными пользователь обязательно
должен указать сумму, с которой будет сдача.

Уточнять время самовывоза

Поставьте галку, если пользователь обязательно должен указать,
через сколько минут он заберёт заказ.

Требовать принять оферту

Поставьте галку, если пользователь обязательно должен принять оферту.
